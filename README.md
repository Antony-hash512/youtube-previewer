# YouTube Previewer

A simple local tool to preview how your video title and thumbnail will look on YouTube. It simulates both the **Sidebar (Suggested)** view and the **Home Page** view.

## Features
- **Multi-Slot Comparison**: Compare up to 3 different thumbnails/titles side-by-side.
- **Real-time Preview**: Instantly see changes as you type.
- **Thumbnail Support**:
    - Paste an image URL.
    - **Drag and Drop**: Drag an image directly into a specific slot's drop zone.
    - **Open File**: Use the file picker for each slot.
- **Duration Toggle**: Check "Show Hours" to switch between `HH:MM:SS` and `MM:SS` duration formats.
- **Copy/Paste**: Quickly copy titles between slots using dedicated buttons.

## How to Use
1. Open `previewer.html` in any modern web browser.
2. **Shared Controls**: Set the **Channel Name** and **Time Format** at the top (applies to all slots).
3. **Slots 1, 2, 3**:
    - Enter a unique **Title** for each slot.
    - Load an image (URL, File, or Drag & Drop) for each slot.
4. You can see the comparison:
    - **Sidebar**: Stacked vertically.
    - **Home Page**: Arranged horizontally for side-by-side comparison.
