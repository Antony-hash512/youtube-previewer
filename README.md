# YouTube Previewer

A simple local tool to preview how your video title and thumbnail will look on YouTube. It simulates both the **Sidebar (Suggested)** view and the **Home Page** view.

## Features

- **Real-time Preview**: Instantly see changes as you type the title or channel name.
- **Thumbnail Support**:
    - Paste an image URL.
    - **Drag and Drop**: Drag an image file directly onto the page.
    - **Open File**: Click the "Open File" button to select an image from your computer.
- **Two Layouts**:
    - **Sidebar**: How it looks in the "Suggested Videos" list.
    - **Home**: How it looks on the YouTube homepage or search results.

## How to Use

1. Open `previewer.html` in any modern web browser.
2. Enter your video **Title** and **Channel Name**.
3. Add a thumbnail:
    - Paste a URL into the "Image URL / Path" field.
    - OR click "Open File" to upload an image.
    - OR drag and drop an image anywhere on the page.
4. The preview updates automatically.
